# Sistema_Passagem_Aerea
Aplicação em Java baseada em Programação Orientada a Objetos para gerenciamento de linhas aéreas. Utiliza encapsulamento para proteger dados de passageiros e herança para estruturar diferentes tipos de voos. O sistema automatiza a consulta de rotas, controle de assentos e emissão de bilhetes de forma segura.
