# ✈️ Sistema de Passagens Aéreas

## 📝 Descrição
Aplicação em Java desenvolvida para simular a operação de uma companhia aérea. O sistema gerencia voos, assentos e venda de passagens, utilizando os pilares da Programação Orientada a Objetos para estruturar regras de negócio complexas e garantir a segurança dos dados.

## 🧱 Pilares de POO Aplicados
* **Encapsulamento:** Proteção dos dados dos passageiros e detalhes dos voos através de atributos privados e métodos de acesso (getters/setters) validados.
* **Herança:** Reutilização de código através de classes como `VooNacional` e `VooInternacional`, que herdam características de uma classe base `Voo`.

## 🚀 Funcionalidades
* Cadastro e consulta de passageiros.
* Listagem de voos com controle de disponibilidade de assentos.
* Compra de passagens com cálculo automatizado de taxas/descontos.

## 🛠️ Tecnologias Utilizadas
* **Java SE**
* **Paradigma:** Orientação a Objetos (POO)

## 💻 Como Executar
1. Clone o repositório.
2. Navegue até a pasta do projeto.
3. Abra em sua IDE de preferência.
4. Execute a classe principal (`Main.java`).
